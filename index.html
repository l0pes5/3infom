Versionamento

O Git é um software que permite realizar versões de um projeto. O Github Desktop é um software que faz a ligação entre VD Code, Git e Github.

Github Desktop permite fazer a sincronia entre o repositório local e o repositório remoto.

Commit

É o processo de confirmação de uma versão entre vígulas, ou seja, a cada interação é possivel confirmar a alteração.

Push

Este comando envia(sincroniza) alterações no repositório local com o repositório remoto no Githib.
